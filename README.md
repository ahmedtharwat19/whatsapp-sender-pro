# WhatsApp Sender Pro

برنامج احترافي لإرسال رسائل WhatsApp تلقائياً مع نظام ترخيص متقدم.

## المميزات

1. إرسال رسائل WhatsApp تلقائياً
2. دعم ملفات Excel، CSV، JSON
3. إضافة نص على الصور مع تخصيص اللون والموضع
4. نظام ترخيص متقدم مع خطط أسعار
5. معرف جهاز فريد غير قابل للتغيير
6. تشفير قوي لجميع البيانات الحساسة
7. واجهة مستخدم متعددة اللغات
8. إدارة التراخيص عبر Firebase

## التثبيت

### المتطلبات الأساسية
- Python 3.8 أو أعلى
- Chrome أو Chromium مثبت على النظام
